🎮 Guess the Number Game 🎲

Guess the number between 0-100 within 7 chances!

How to Play:

👉🏻 You need to guess a number between 0 and 100.

👉🏻 You have 7 chances to guess the correct number.

👉🏻 After each guess, you will be informed if your guess is correct or not.

👉🏻 The game will end when you either guess the correct number or run out of chances.

👉🏻 Good luck and have fun!
